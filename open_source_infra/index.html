Open Source Infrastructure List

I work as a Systems Engineer on the OpenStack Infrastructure team. We have built the open source infrastructure for the open source project, everything we touch is open source! So the lucky engineers on our team get to work on an awesome project and travel the world and tell people about it. In the course of these travels, we've met several teams also doing the same thing.

The following is a list of open source infrastructures that we're familiar with:

  • OpenStack: http://docs.openstack.org/infra/system-config/ (the one I work on)
  • Debian:
    • https://dsa.debian.org/ (Debian System Administrators team)
    • https://ci.debian.net/ (Continuous Integration)
  • Mozilla: https://wiki.mozilla.org/ReleaseEngineering/PuppetAgain

FAQ

Q: What is the criteria for inclusion on this site?

A: It must be an open source project with an open source infrastructure. Projects that operate a fully open source infrastructure are preferred and may soon be split into a specific section. However, if the project has a partially open source infrastructure that may be valuable to other projects and organizations, that may be added as well.

Q: This is a site about Open Source Infrastructures, but you host the code for this site on GitHub, a proprietary platform. Doesn't that make you feel bad and silly?

A: Yes.
